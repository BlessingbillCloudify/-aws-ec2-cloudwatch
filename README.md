# AWS EC2 with Terraform and CloudWatch Monitoring

This project demonstrates how to provision an AWS EC2 instance using Terraform and monitor it with Amazon CloudWatch.

## Technologies Used
- Terraform
- AWS EC2
- AWS CloudWatch
- Git & GitHub

## Project Overview
- Provisioned an EC2 instance using Terraform
- Configured security groups
- Enabled monitoring using CloudWatch
- Managed infrastructure as code (IaC)

## Use Case (Real-Life Scenario)
A company wants to deploy a virtual server quickly and monitor its performance.
Terraform automates the infrastructure creation, while CloudWatch helps track CPU usage and system health.
